PROJECT OVERVIEW
================================================================================
TSVaultKeySafe is a privacy-first, offline-only, end-to-end encrypted digital 
vault for securely storing product licenses, serial numbers, receipts, and 
warranty documents.

SECURITY SPECIFICATIONS
================================================================================
Encryption
  - Algorithm: AES-256-GCM (NIST-approved)
  - Key Derivation: HKDF-SHA256
  - PIN Hashing: PBKDF2-SHA256 (100,000 iterations)
  - Nonce: 96-bit random per operation
  - Authentication Tag: 128-bit

Key Management
  - Master Key: Platform keystore (Android Keystore / iOS Keychain)
  - Database Key: Derived on-demand from master key
  - Attachment Key: Derived on-demand from master key
  - Export Key: Derived from user passphrase

Authentication
  - PIN: 6-digit numeric with rate limiting
  - Biometric: Delegated to OS-level security
  - Rate Limiting: 3 failed attempts → 30-second lockout
  - Auto-Lock: Configurable timeout (1min / 5min / 15min / Never)

HOW TO USE THIS REPOSITORY
================================================================================

1. CLONE THE REPOSITORY
   git clone https://github.com/tsvaultkeysafe/app.git
   cd tsvaultkeysafe

2. INSTALL DEPENDENCIES
   pnpm install

3. START DEVELOPMENT SERVER
   pnpm start

4. RUN ON DEVICE
   pnpm ios      # iOS Simulator
   pnpm android  # Android Emulator
   pnpm web      # Web Browser

5. RUN TESTS
   pnpm test

6. BUILD FOR PRODUCTION
   eas build --platform android
   eas build --platform ios

7. SUBMIT TO APP STORES
   eas submit --platform android
   eas submit --platform ios
